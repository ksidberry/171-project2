Tweet Map

Description:

The above visualization is a data bubble map that displays tweets by location. The color of the bubbles represent the sentiment readings of that particular tweet. The legend is presented above. The size of the bubbles are determined the tweet's klout, or its influence in the Twitter space. The tweets latitude and longitude is used to geographically position the bubble on the map. The text, location, temperature, and a sentiment values of the tweet are included in the tooltip when the user hovers over the tweet bubbles.
